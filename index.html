📅 Создание календаря в Microsoft Project

🎯 Цель

В этом руководстве показано, как создать новый календарь, изменить рабочее время и использовать его в проекте.

🧩 Шаг 1: Открытие настроек календаря

Сперва перейдите на вкладку Project, затем найдите кнопку Change Working Time.

[Скриншот: вкладка Project → Change Working Time]

🧩 Шаг 2: Создание нового календаря

После нажатия откроется окно. Нажмите Create New Calendar..., введите имя календаря и выберите Create new base calendar.

[Скриншот: создание нового календаря]

🧩 Шаг 3: Выбор календаря

Далее выберите созданный календарь в списке For calendar: (Ваш календарь).

[Скриншот: выбор календаря]

🧩 Шаг 4: Настройка времени

Задайте начальную и конечную даты работы. Для более глубокой настройки нажмите Details и установите рабочие часы.

[Скриншот: настройка времени]

📊 Использование календаря

После настройки календаря его можно применять к задачам проекта. Календарь влияет на расписание, длительность задач и рабочие дни.
